### 2021/11/7

+ 长沙—南昌地区取消了桥，改为老版山地，适合大部队展开。
+ 城市名美化、添加了地标名
+ 路障生命值-50%
+ 装甲攻势修改为步坦协同，坦克护甲保留，步兵护甲转移至摩托化，坦克附近步兵溅射减免10%，护甲+1，伤害+15%
+ 沙袋范围略微增加，并且提供0.05射程
+ 修复了一些坦克选择半径过小的问题
+ 添加了-light x命令，x=1，2，3到6. 修改你的游戏光照模式默认1(风和日丽)
+ 美舰第一波略微减少数量，开战时日舰除航母自动归于南线
+ 火炮移速降低，日本火炮将领改成射程+0.5，射速+10%，日本开局添加105科技，火炮科研时间降低，但是需要点重视装甲国策
+ 大幅提高了坦克炮对轻甲的伤害，pe坦克量略减
+ 取消军火的人力要求
+ 删除了帝国重工业的debuff
+ 强化了各类机枪兵的武器伤害、攻速、减甲量，并且不会攻击碉堡，削弱了陈诚。
+ 工业规划技能改为50pp +1/+5
+ 王世杰效果加强到200%
+ 海军不再需要人力，荷兰船舶公司修改为速度，山本效果修改为生命
+ 战列可以进广州湾，巨舰效果改为伤害
+ 战舰转身速度提高，必须先转身再移动，符合物理学
+ 略微减少了山西、徐州的初始防御
+ pve广东英国图纸瞬间研发
+ 修复了日本山地将的问题
+ 万岁冲锋速度提高20%
+ 本子4枪射程回调0.1，抗战4级以下步枪射程均降低0.1
+ 三光政策效果增强5级250，4级150，3级100
+ 全局免费GDP增长机制对PVP不再生效，PVE起100%作用
+ 驱使财阀修改为获得一个免费GDP增长机制，每分钟+5，工厂价格-15
+ 皖南事变现在使得校长太祖获得-0.1的经济和人力减少，-5pp收入，不再取消视野和结盟
