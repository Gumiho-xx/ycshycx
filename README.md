### 自定义国策顺序系统：
##### 直接下载上方的word攻略，有详解


飞机现在不会追射程外的空军单位。<n/>日本小航母采用更加昭和风的新皮肤。放大了小船模型。添加了大船的移动水纹。加大了战舰半径，海战不会太过剧团。中线添加一个大和级战列舰，1500血500盾50甲240攻6射程。当联合舰队沉没时才能玩。拥有技能菊水特攻，什么玩意都能撞沉。<n/>选人界面更改：添加随机选人模式。10人有5人要求随机就必须随机(pvp和逆向pve只需要3个人要求)。添加了3个私人选项。<n/>更新了城市无敌的算法，由原来的单位行为修改为城市自身行为(可能会缓解卡顿，目测fps+10%)。删除地图上所有自带的沙袋(目测fps+2%)<n/>添加了报播员和种族选择功能(就是个控制台和语音包功能)。<n/>无人的城市旗帜更改为风莉大小姐，Kazari我的菜呜呜呜。<n/><n/>日本火炮将领改为气球射程buff+2(初始1)，气球视野+2<n/>所有火炮初始对建筑伤害略增。<n/>北线一号作战国策直接改为出限时AI系统兵进攻华北防线。提前完成会在开战瞬间出来。<n/>移除徐州南昌长沙的工厂。<n/>防空火箭车由12分钟cd延长到16分钟cd(对抗战pvp)。<n/>抗战科技惩罚由40%提到(50%火炮75%坦克100%飞机100%海军)。气球现在也会吃这个debuff。<n/>步枪-机枪科技线整合。N级机枪由N-1级机枪+N级步枪解锁，N级机枪研发时间大幅降低。<n/>万岁模式下，攻速30%，50%提高到50%，80%。移速由70%，100%提到100%，150%。<n/>略微降低了城市的gdp。为了不影响pe，pe种地价格-10。<n/>加强了神风特攻的伤害，尤其是对船的攻击提到500真伤。<n/>清洗皇道派现在会+80pp，不再给人。三线初始人力+50。<n/>最后一搏国策移除，改为决议。美国参战后可用，每用一次，立即+50人力，单位生命值减少10%。<n/>抗战+20胜利点临界由莱特湾前置到虎。<n/>黄金十年效果改为永久。乡村建设不再针对低级城市。<n/>日本反坦克伤害回调，与抗战相同。<n/><n/>意志至上调整为意志至上与科技至上两个互斥国策。<n/>意志至上：需要3级素质的前置。直接为+15血和常驻2甲。日本素质训练每级提升15(由10上调)。<n/>科技至上：研发速度+25%，带有原先陆军扩编法案的效果(减少高科技人力)+1个重工流水线+1个飞机流水线效果，其前置为军工国有化。原先的陆军扩编法案调整为1个枪支流水线+1个重工流水线的效果。自动国策下，会点意志，科技代替原本的最后一搏成为49号国策。<n/><n/>为了使淞沪会战刺激一点，全面体验海陆空一体化作战。太湖方向调整，淞沪初始防线加强，直接送到上海最前线。中线开局一战列换一航母。登陆准备再获得一个120秒限制时间的航母。校长开局补1个战斗机。为防南线带3个航母干预淞沪，航母国策需要开战后才能点。另外长沙pvp下的2个机枪堡不再移除。
